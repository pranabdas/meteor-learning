# Meteor Learning

## Setup

Current version of Meteor requires node v14. You may use `nvm`:

```console
nvm install 14
nvm use 14
```

Install meteor globally:

```console
npm install -g meteor
```

Initialize new meteor project:

```console
meteor create meteor-learning
```

Start development server:

```console
meteor run
```

or simply:

```console
meteor
```
